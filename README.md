# RL Blackjack DQN

## Descripción
Este proyecto implementa un agente de Aprendizaje por Refuerzo usando Deep Q-Network (DQN) para el entorno Blackjack de Gymnasium.

## Componentes
- Red neuronal (PyTorch)
- Estrategia epsilon-greedy
- Replay Buffer
- Entrenamiento con batches
- Evaluación del agente

## Ejecución

```bash
python src/train.py
